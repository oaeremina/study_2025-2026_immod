## Модель SIR

Уравнения модели:
dS/dt = -β·I·S/N
dI/dt = β·I·S/N - γ·I
dR/dt = γ·I

Параметры: β = 0.4, γ = 0.1
Начальные условия: N = 1000, I0 = 5

Результаты:
Базовое репродуктивное число R0 = 4.0
Пик эпидемии: 258 человек на 38 день
Всего переболело: 798 человек

## Модель Лотки-Вольтерры

Уравнения модели:
dx/dt = α·x - β·x·y
dy/dt = δ·x·y - γ·y

Параметры: α=0.1, β=0.02, δ=0.01, γ=0.3
Начальные условия: x0=40 (жертвы), y0=9 (хищники)

Стационарные точки:
x* = 30.0, y* = 5.0

Период колебаний жертв: 37.2 ± 1.5 ед. времени
Период колебаний хищников: 37.2 ± 1.5 ед. времени

Статистика жертв: мин=11.2, макс=83.5, среднее=40.2
Статистика хищников: мин=2.1, макс=12.8, среднее=5.1
