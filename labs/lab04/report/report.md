# Теоретическое введение


## Модель SIR

Модель SIR, предложенная Кермаком и Маккендриком в 1927 году, описывает динамику эпидемии в популяции, разделённой на три группы:

| Обозначение | Название | Описание |
|-------------|----------|----------|
| **S** | Susceptible (Восприимчивые) | Здоровые индивиды, которые могут заразиться |
| **I** | Infectious (Инфицированные) | Больные индивиды, способные заражать |
| **R** | Recovered (Выздоровевшие) | Переболевшие, приобретшие иммунитет |

Классическая модель на обыкновенных дифференциальных уравнениях имеет вид:

$$ \frac{dS}{dt} = -\frac{\beta S I}{N}, \quad \frac{dI}{dt} = \frac{\beta S I}{N} - \gamma I, \quad \frac{dR}{dt} = \gamma I $$

где:
- $\beta$ — коэффициент заразности (скорость передачи инфекции),
- $\gamma$ — скорость выздоровления ($1/\gamma$ — средняя продолжительность болезни),
- $N = S + I + R$ — общая численность популяции.

## 5. Сравнение с теоретической моделью

Для проверки корректности агентной модели сравним полученные результаты с теоретическими ожиданиями классической модели SIR.

**Теоретические формулы:**

$$ R_0 = \frac{\beta}{\gamma}, \quad \text{где } \gamma = \frac{1}{\text{infection\_period}} = \frac{1}{14} \approx 0.0714 $$

**Расчёт R₀ для различных β:**

| β | R₀ | Прогноз (эпидемия?) | Наблюдение (пик > 5%) |
|---|----|--------------------|----------------------|
| 0.1 | 1.4 | Да | Нет (2%) |
| 0.2 | 2.8 | Да | Да (8%) |
| 0.3 | 4.2 | Да | Да (18%) |
| 0.5 | 7.0 | Да | Да (45%) |
| 1.0 | 14.0 | Да | Да (75%) |

**Наблюдение:** Порог возникновения эпидемии в агентной модели выше теоретического (R₀ > 2 вместо R₀ > 1). Это объясняется:
- Пространственной структурой (инфекция начинается в одном городе).
- Стохастическими эффектами (инфекция может случайно затухнуть при малом числе инфицированных).

---

### 6.1 Определение базового репродуктивного числа

Для параметров по умолчанию (β = 0.5, γ = 1/14 ≈ 0.0714):

$$ R_0 = \frac{\beta}{\gamma} = \frac{0.5}{0.0714} \approx 7.0 $$

Это означает, что в полностью восприимчивой популяции один больной в среднем заражает 7 человек.

### 6.2 Порог возникновения эпидемии

Минимальное значение β, при котором пик инфицированных превышает 5% популяции, составляет ≈ 0.2. Теоретический порог R₀ = 1 соответствует β = γ = 0.0714. Расхождение объясняется малым начальным числом инфицированных (1 человек) и стохастичностью.

### 6.3 Эффект гетерогенности

При задании разных значений β для разных городов (например, β₁ = 0.5, β₂ = 0.3, β₃ = 0.1) динамика становится асимметричной: инфекция быстрее распространяется в городах с высокой заразностью, но через миграцию передаётся в города с низкой заразностью.

3. **Гетерогенность:** Разная заразность в разных городах приводит
